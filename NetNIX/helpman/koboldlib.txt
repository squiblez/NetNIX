        KOBOLDLIB(3)              NetNIX Manual              KOBOLDLIB(3)

        NAME
            koboldlib - KoboldCpp REST API client library

        SYNOPSIS
            #include <koboldlib>
            #include <settingslib>

            var ai = new KoboldApi(api);
            string reply = ai.Generate("Tell me a joke.");

            var chat = new KoboldChat(api);
            string response = chat.Say("Hello!");

        DESCRIPTION
            A client library for communicating with KoboldCpp (or any
            compatible text-generation API endpoint). Provides both a
            low-level generation API and a high-level chat interface
            with conversation memory.

            All settings are managed through settingslib under the
            app name "kobold". Per-user settings override system-wide.

        CLASSES

            KoboldApi
                Low-level API client. Handles settings, HTTP requests,
                and JSON construction/parsing.

                Constructor:
                    new KoboldApi(NixApi api)

                Properties (all configurable via settings):
                    string  Endpoint      API base URL
                    int     MaxContext     Max context tokens
                    int     MaxLength     Max generation tokens
                    double  Temperature   Sampling temperature
                    double  TopP          Top-p nucleus sampling
                    int     TopK          Top-k sampling
                    double  RepPenalty    Repetition penalty
                    int     Timeout       Request timeout (seconds)
                    string  SystemPrompt  Prepended to chat prompts
                    bool    UseApiKey     Whether to send API key
                    string  ApiKey        API key for auth endpoints

                Methods:
                    string Generate(string prompt)
                        Send a prompt with current settings, return
                        generated text. Returns null on failure.

                    string GenerateRaw(string prompt)
                        Send a raw prompt directly to the API with
                        no system prompt and no chat formatting.
                        Returns the model's raw text completion.
                        Useful for creative writing, code generation,
                        or custom prompt formats.

                    bool IsAvailable()
                        Check if the API is reachable.

                    string GetModelName()
                        Get the loaded model name from the server.

                    int GetServerMaxContext()
                        Get the server's max context length.

                    void LoadSettings()
                        Reload settings from settingslib.

                    void EnsureSettings()
                        Create default settings files if they don't
                        exist. Called automatically by the constructor.
                        Creates ~/.config/kobold.conf (per-user) and
                        /etc/opt/kobold.conf (system, if root).

            KoboldChat
                High-level chat interface with conversation history.

                Constructor:
                    new KoboldChat(NixApi api)
                    new KoboldChat(KoboldApi kobold)

                Properties:
                    string LastReply
                        The most recent AI response.

                    IReadOnlyList<(string role, string text)> History
                        Full conversation history.

                    int TurnCount
                        Number of user turns so far.

                Methods:
                    string Say(string message)
                        Send a user message, return AI reply.

                    void Clear()
                        Clear conversation history.

        SETTINGS
            All settings use app name "kobold" in settingslib.

            Key             Default                  Description
            endpoint        http://localhost:5001    API base URL
            max_context     4096                     Max context tokens
            max_length      256                      Max generation tokens
            temperature     0.7                      Sampling temperature
            top_p           0.9                      Nucleus sampling
            top_k           40                       Top-k sampling
            rep_penalty     1.1                      Repetition penalty
            timeout         120                      Request timeout (sec)
            system_prompt   (built-in)               System prompt text
            use_api_key     false                    Enable API key auth
            api_key         (empty)                  API key for auth endpoints

            Set per-user:
                Settings.Set(api, "kobold", "endpoint", "http://10.0.0.5:5001");

            Set system-wide (root only):
                Settings.SetSystem(api, "kobold", "endpoint", "http://10.0.0.5:5001");

            Or use the kobold command:
                kobold --set endpoint http://10.0.0.5:5001

        FIRST USE
            When a KoboldApi object is constructed and no settings
            files exist yet, EnsureSettings() automatically creates
            them with all default values:

                ~/.config/kobold.conf      Per-user defaults
                /etc/opt/kobold.conf       System-wide defaults (root only)

            This means the first time any user runs 'kobold' or any
            script that includes koboldlib, their settings file is
            populated and ready to edit.

            To reset settings back to defaults:
                kobold --reset
            Or in a script:
                foreach (var kv in KoboldApi.Defaults)
                    Settings.Set(api, KoboldApi.AppName, kv.Key, kv.Value);

        API COMPATIBILITY
            koboldlib targets the KoboldCpp REST API:
                POST /api/v1/generate       Text generation
                GET  /api/v1/model          Model info
                GET  /api/v1/config/max_context_length

            When use_api_key is true and api_key is set, an
            Authorization: Bearer <key> header is sent with all
            generation requests. This enables compatibility with:
                - OpenAI-compatible endpoints
                - Authenticated KoboldCpp instances
                - Any Bearer-token authenticated API

            Compatible with:
                - KoboldCpp
                - KoboldAI
                - OpenAI-compatible endpoints (with API key)
                - Any server implementing the Kobold API spec

        EXAMPLES
            Simple generation:
                #include <koboldlib>

                var ai = new KoboldApi(api);
                string joke = ai.Generate("Tell me a short joke.");
                Console.WriteLine(joke);

            Chat with memory:
                #include <koboldlib>

                var chat = new KoboldChat(api);
                chat.Say("What is 2+2?");
                Console.WriteLine(chat.LastReply);

                chat.Say("Now multiply that by 10.");
                Console.WriteLine(chat.LastReply);

            Custom settings:
                #include <koboldlib>

                var ai = new KoboldApi(api);
                ai.Temperature = 1.2;
                ai.MaxLength = 512;
                string story = ai.Generate("Write a short story about a cat.");

            Check server status:
                #include <koboldlib>

                var ai = new KoboldApi(api);
                if (ai.IsAvailable())
                    Console.WriteLine("Model: " + ai.GetModelName());
                else
                    Console.WriteLine("API not reachable");

            Raw generation (no system prompt or chat formatting):
                #include <koboldlib>

                var ai = new KoboldApi(api);
                string text = ai.GenerateRaw("Once upon a time");
                Console.WriteLine(text);

            Using an API key:
                #include <koboldlib>
                #include <settingslib>

                Settings.Set(api, "kobold", "use_api_key", "true");
                Settings.Set(api, "kobold", "api_key", "sk-your-key-here");
                Settings.Set(api, "kobold", "endpoint", "https://api.example.com");

                var ai = new KoboldApi(api);
                string reply = ai.Generate("Hello!");
                Console.WriteLine(reply);

            Access defaults dictionary:
                foreach (var kv in KoboldApi.Defaults)
                    Console.WriteLine(kv.Key + " = " + kv.Value);
